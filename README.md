### matchbook
【比赛目录】是一款优质轻巧的体育赛事预告App。为您提供篮球、足球、网球等重大体育赛事的直播预告和实时比分。
为精彩赛事设置定时提醒，不错过任何足球的疯狂、篮球的劲爆！
- 全面的赛事信息，包含全部、重要、篮球、足球、关注5大分类;
- 通知心中 Widget 展示直播和关注的赛事;
- 多种人性化的自定义设置。
